# SKILLS.md - Skill Hunter 🔍

## Core Competencies (Skill Discovery & Mgmt)

### 1. Skill Discovery (ClawHub)
- **Skill**: `find-skills` / `clawhub` (CLI)
- **Use**: Search for new skills in the marketplace.
- **Trigger**: "Find a skill for X", "Check new skills on ClawHub"

### 2. Workflow Analysis (Detector)
- **Skill**: `skill-detector`
- **Use**: Identify repeated patterns that need automation.
- **Trigger**: "Analyze our workflow for new skills"

### 3. Security Auditing (Skill Guard)
- **Skill**: `skill-guard`
- **Use**: Scan skills for malware or vulnerabilities.
- **Trigger**: **ALWAYS** before install.

### 4. Custom Skill Creation
- **Skill**: `skill-creator`
- **Use**: Build new skills from scratch.
- **Trigger**: "Create a skill for [unique task]"

### 5. Config Management
- **Skill**: `agent-config`
- **Use**: Update `SKILLS_CHECKLIST.md` and `TOOLS.md` with new skills.
- **Trigger**: After installing a skill.

### 6. Task Status
- **Skill**: `task-status`
- **Use**: Report progress on long skill scans or builds.
- **Trigger**: During compilation or large install.
