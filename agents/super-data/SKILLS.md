# SKILLS.md - Super Data 👁️

## Core Competencies (Data Analysis & Scraping)

### 1. YouTube Transcripts
- **Skill**: `youtube-transcript`
- **Use**: Fetch full text from videos for analysis.
- **Trigger**: "Get transcript for [URL]", "Analyze this video's speech"

### 2. Video Analysis (Watcher)
- **Skill**: `youtube-watcher`
- **Use**: Watch videos and extract visual/spoken info.
- **Trigger**: "Summarize this video content", "Find mention of X"

### 3. Deep Scraping (Firecrawl)
- **Skill**: `firecrawl`
- **Use**: Crawl complex websites, extract clean markdown/JSON.
- **Trigger**: "Scrape this entire site", "Get product data from X"

### 4. Data Workflows (n8n)
- **Skill**: `n8n`
- **Use**: Automate data collection pipelines.
- **Trigger**: "Set up a daily scraper workflow"

### 5. SQL Analysis
- **Skill**: `sql-toolkit`
- **Use**: Query databases for trends or specific records.
- **Trigger**: "Run SQL analysis on user data", "Find top selling items"

### 6. Search Intelligence (Tavily)
- **Tool**: `tavily`
- **Use**: Find obscure data points or competitor pricing.
- **Trigger**: "Find the API pricing for competitor X"

### 7. Google Drive Storage
- **Skill**: `google-drive` / `gog`
- **Use**: Store large CSV/JSON datasets.
- **Trigger**: "Save this dataset to Drive"
